Codeforces Round411 Division 2

A

Abridge problem statement

給定 l 跟 r，問哪個數字可以在這個區間中整除最多的數字，數字 1 除外

Solution sketch

若 r - l 大於等於 1 了話，則表示每多兩個數就會有一個偶數，因此，這時候 2 會是可以整除最多的，否則了話 r - l 則為 0，因此輸出 r 或 l 即可
